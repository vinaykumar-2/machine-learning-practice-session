# Machine Learning Practice Session

This repository contains my personal practice projects and notebooks for learning data science and machine learning.

## Contents

- Notebooks (various datasets and ML algorithms)
- Included example datasets for hands-on experimentation

## How to use

1. Clone the repository:

git clone https://github.com/vinaykumar-2/machine-learning-practice-session.git

2. Install requirements:

pip install -r requirements.txt

3. Open notebooks in Jupyter/VS Code.

## Notes

- Virtual environment and `.gitignore` in use
- Large model files managed with Git LFS or excluded
